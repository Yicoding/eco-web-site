# nvm

- `nvm`（node.js version management）是一个`nodejs的版本管理工具`。nvm 和 n 都是 node.js 版本管理工具，为了解决 node.js 各种版本存在不兼容现象可以通过它可以安装和切换不同版本的 node.js

## 1.查看本地安装的所有版本

```bash
nvm list
```

## 2.安装不同版本的 node

```bash
nvm install 11.13.0
```

## 3.使用特定版本

```bash
nvm use 11.13.0
```

## 4.卸载

```bash
nvm uninstall 11.13.0
```
