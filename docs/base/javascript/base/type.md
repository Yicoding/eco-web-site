> 分为基本数据类型和引用数据类型

`1.基本类型`

| 分类      |     名称     |
| :-------- | :----------: |
| String    |    字符串    |
| Number    |     数字     |
| Boolean   |     布尔     |
| Null      |      空      |
| Undefined |    未定义    |
| Symbol    | 独一无二的值 |
| BigInt    |    大整数    |

`2.引用类型`

| 分类     | 名称 |
| :------- | :--: |
| Object   | 对象 |
| Array    | 数组 |
| Function | 函数 |

`3.JS数据类型`

> `String`、 `Number`、 `Boolean`、 `Null`、 `Undefined`、 `Object` 以及 ES6 新加的 `Symbol`和`BigInt`
