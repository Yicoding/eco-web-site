# 正则

用来描述规则的表达式

## 1.字符

### 1）单个字符

| 正则 | 特殊字符   | 记忆方式                                  |
| :--- | :--------- | :---------------------------------------- |
| \n   | 换行符     | new line                                  |
| \s   | 空白符     | space                                     |
| \f   | 换页符     | form feed                                 |
| \r   | 回车符     | return                                    |
| \t   | 制表符     | tab                                       |
| \v   | 垂直制表符 | vertical tab                              |
| [\b] | 回退符     | backspace,之所以使用[]符号是避免和\b 重复 |

### 2）多个字符

| 正则 | 匹配区间                 | 记忆方式            |
| :--- | :----------------------- | :------------------ |
| .    | 除了换行符之外的任何字符 | 句号,除了句子结束符 |
| .    | 除了换行符之外的任何字符 | tab                 |
| .    | 除了换行符之外的任何字符 | tab                 |
| .    | 除了换行符之外的任何字符 | tab                 |
